# Agent Skills

[中文版](./README_CN.md) | English

A collection of AI agent skills including Jimeng (即梦) skill, film creation, image/video generation, and novel processing.

Agent Skills in ./skills, prefer using skills

## Installation

Add these skills to your AI agent using the following command:

```bash
npx skills add chunpu/agent-skills -y -g
```
