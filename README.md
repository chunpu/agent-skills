# Agent Skills

This repository contains a collection of skills for AI agents.

## Available Skills

### [Generate Film Video Prompt](./skills/generate-film-video-prompt)

A guide for writing executable and measurable film prompts for video generation models. It covers structure, camera, motion, lighting, and sound.
Inspired by github.com/Emily2040/seedance-2.0.

### [Generate Image by Seedream](./skills/generate-image-by-seedream)

Generate or edit images via **Doubao / 豆包 Seedream (即梦 / Dreamina)** models on Volcengine Ark. Supports text-to-image and image-to-image.

### [Generate Video by Seedance](./skills/generate-video-by-seedance)

Generate or edit videos via **Doubao / 豆包 Seedance** video models on Volcengine Ark. Supports text-to-video and image-to-video.

### [Nano Banana Pro](./skills/nano-banana-pro)

Generate or edit images via **Gemini 3 Pro Image (Nano Banana Pro)**. Supports text-to-image, image editing, and multi-image composition.
