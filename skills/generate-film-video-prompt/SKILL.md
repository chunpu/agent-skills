# 通用影视提示词 · 中文核心 Skill

> 目标：用一份 skill 快速写出**可执行、可测量**的影视提示词，适配任意视频生成模型。  
> 输出必须：可被摄像机、光表、秒表验证；所有引用都说明角色；尽量用中文影视词汇。

---

## 1. 总体结构：五层 + 两条铁律

**铁律 1：前 20–30 个字必须是「主体 + 核心动作」**  
**铁律 2：每个 clip 只允许 1 个主动作（动词）**

推荐结构（六件事）：

1. **SHOT**：景别/构图（远景/中景/近景/特写…）
2. **SUBJECT**：谁（角色、物体）、外形特征
3. **ACTION**：做什么（1 个主动作 + 简短物理/节奏）
4. **CAMERA**：镜头 framing + 运动 + 速度 + 视角
5. **STYLE / LIGHT**：风格与基调（镜头语言+灯光，而不是形容词）
6. **SOUND**：环境音 + SFX + 音乐/静音 决策

一句话写法示例（模板）：

> `中景，[主体]，[动作]，[环境]，[镜头运动+时长]，[灯光+色调]，[声音设计]。[长宽比][秒数]`

---

## 2. @Tag 使用：所有资源都要“有职位”

请根据你所用平台/模型的**资源限制**（图片/视频/音频的数量与大小）来规划输入。  
经验上，单次生成中图片、视频、音频总数控制在**十几个以内**更稳定。

**每个 @Tag 必须说明角色：**

- `@Image1 作为主角形象参考`
- `@Image2 作为场景/环境参考`
- `@Image3 作为首帧画面`
- `@Video1 的运镜作为整个片子的镜头风格参考`
- `@Audio1 作为节奏和情绪参考`

**“参考” vs “编辑”区别：**

- 参考（生成新内容）：`参考 @Video1 的运镜和剪辑节奏`
- 编辑（改原视频）：`将 @Video1 中的人物替换为 @Image1 的角色`

---

## 3. 镜头（Camera）：四要素合同

每个镜头都要给出这 4 项：

- **Framing**：`远景 / 全景 / 中景 / 近景 / 特写 / 过肩 / POV`
- **Movement**：`固定 / 推镜 / 拉镜 / 平移 / 摇镜 / 仰俯 / 环绕 / 跟拍 / 升降 / 手持`
- **Speed / Duration**：`缓慢 / 适中 / 快速` + `在 X 秒内完成`
- **Angle**：`平视 / 低角度 / 高角度 / 俯视 / 上帝视角 / 荷兰角`

可直接复制的句式：

- `固定机位，镜头完全不移动`
- `缓慢推镜，从中景推进到面部特写，耗时 8 秒`
- `缓慢拉镜，拉远到全景，交代完整环境`
- `水平摇镜跟随主角从左到右`
- `低角度仰拍，突出角色压迫感`
- `环绕主角顺时针缓慢移动，保持距离不变`
- `手持跟拍，轻微抖动但不混乱`

多镜头在一个 clip 里写法（2–3 个镜头为宜）：

> `镜头1：远景……。[切到] 镜头2：中景……。[切到] 镜头3：近景……。环境音保持连续。`

## 4. 运动（Motion）：节奏与时间语言

**时长 vs 变化次数：**

- 4–6s：1 个明显变化
- 8–10s：1–2 个变化
- 12–15s：2–3 个变化（建议用时间段分段：`0–4s / 4–8s / 8–END`）

时间写法：

- `0–3 秒：……；3–6 秒：……；6–10 秒：……`
- `动作在 2 秒内缓慢加速，在第 3 秒突然停下`
- `最后 0.8 秒画面保持静止`

**静止合约（避免乱抖）：**

- `主体保持基本静止，只能看到轻微呼吸`
- `禁止机位漂移`
- `风很小，只带动一点衣物`

---

## 5. 灯光与氛围（Lighting & Atmosphere）

**总规则：一定要点名光源。**

对比：

- ✗ `戏剧化灯光`  
- ✓ `单侧 45° 主光从镜头左侧打来，3200K 暖光，右侧基本无填充`

常用灯光描述：

- `窗户背光，长条影子朝镜头方向拉长`
- `头顶单颗白炽灯作为主光，形成眼眶阴影`
- `街边路灯作为主光，背景霓虹做轮廓光`
- `烛光在脸上不断跳动，背景几乎全黑`
- `阴天漫射自然光，整体低对比`

氛围（必须可测量）：

- `薄雾，被背光打亮，可见距离约 5 米`
- `轻微飘雨，只在强主光前方明显可见`
- `空气中有大量尘粒，在侧光下清晰可见`
- `寒冷环境，每次呼吸都能看见白雾`
- `地面有轻微热浪，上升形成抖动感`

---

## 6. 风格与材质（Style & Materials）

**风格尽量用“物理/摄影语言”，少用空洞形容词。**

建议控制在 **2–3 个 style token**：

- 镜头感：`变形宽银幕 / 复古软焦 / 数码干净`
- 纹理：`细腻胶片颗粒 / 轻微噪点 / 完全干净`
- 调色：`低饱和 / 暖高光冷阴影 / 霓虹高饱和`
- 对比：`低对比 / 高对比 / 深黑压暗`

材质合约（防止「塑料感」）：

- `拉丝金属，丝光质感，边缘有轻微磨损`
- `旧皮革，完全哑光，纹理清晰，折痕明显`
- `毛衣，表面有细小毛茸，边缘略微起球`

---

## 7. 角色与手部（Characters & Hands）

**角色卡（建议单独写好后复用）：**

> `某某：20 多岁女性，身高约 165cm，中等身材，小麦肤色，短卷发，棕色风衣，眼神锐利，行为冷静克制。`

在提示词中用法：

- `本片主角外观完全参考 @Image1 的角色`
- `角色 A 参考 @Image1，角色 B 参考 @Image2`
- 多人场景：每个动作都指明是谁：  
  `角色 A 向角色 B 走近两步，角色 B 后退一步。`

手部安全：

- 不重要就**尽量不出手**：`从腰部以上取景，手不入画`
- 必要时只给**一个简单动作**：  
  - `右手拿起桌上的杯子`  
  - `左手平摊在桌面`  
  - 避免：`复杂手势 / 近景打字 / 多人手部交叉`

## 8. 特效与破坏（VFX & Destruction）

每个特效给出 5 要素：

1. **Source**：从哪里发出（武器/地面/角色手掌…）
2. **Scale**：大小相对人有多大（手掌大小/车大小/楼大小）
3. **Behavior**：爆发/漂浮/螺旋/扩散/收缩…
4. **Interaction**：是否发光、是否照亮人物/投影/吹动烟尘
5. **Duration**：几秒开始，几秒达到最强，几秒消失

示例：

- `剑与盾碰撞的瞬间，白橙色火花从接触点爆出，沿重力向下飞溅，1 秒内迅速熄灭。`
- `角色周身出现淡蓝色光环，半径约 30 厘米，随呼吸轻微脉动，并在 3 秒内缓慢减弱直至消失。`

破坏物理：

- `冲击点向外延伸的蜘蛛网裂纹，半径约 3 米，裂缝边缘略微翘起。`
- `石墙碎裂成大块石块向下掉落，小碎石以扇形向外喷出 5 米，落地扬起灰尘云。`
- `可见空气冲击波快速向外扩散，半径 10 米 范围内杂物和衣服被明显掀动。`

## 9. 音频与口型（Audio & Lip-sync）

对于**同时生成音频和视频**的模型，音频设计不能忽略。

不同平台/模型支持的音频**格式与时长上限**会略有差异，具体以官方文档为准。  
通用安全做法：

- 使用平台推荐或常见格式（如 **MP3**），码率保持中高（如 ≥128kbps）
- 单段音频控制在 **3–15 秒**，对白类建议 3–8 秒
- 如需多段音频，总时长尽量控制在几十秒以内，避免过长导致失败或不同步

声音四层：

1. 环境音（Ambience）：`雨声 / 风声 / 城市底噪 / 森林虫鸣`
2. 前景音效（SFX）：`脚步、关门、拳头击打、水滴等`
3. 音乐（BGM）：`是否有音乐、何时进、能量高低`
4. 静音设计：`在哪一两秒完全静音，制造张力`

写法示例：

- `声音：全程雨声环境音，远处偶尔传来一声汽车经过。无配乐。`
- `4 秒时加入轻柔钢琴，音量低于环境音。最后 1 秒全部静音，只保留呼吸声。`

文本驱动对白：

- `角色 A 说："我们天亮就出发。"`  
- `镜头：中近景，固定机位。口型自然匹配台词。`

音频驱动口型（@Audio1）：

- `@Audio1 作为对白参考，要求口型严格跟随 @Audio1，从 0 秒到结束。`
- `机位固定，中近景，只允许轻微表情变化，避免大幅度点头/转头。`

**多角色口型：目前官方承认尚未解决 → 实战方案是分角色单独生成 + 后期剪辑合成。**

## 10. 反废话（Anti-slop）与优化顺序

**一条测试：这句话能被“摄像机/光表/秒表”直接测量吗？如果不能，就删或改。**

应当删除的词（典型 slop）：

- `惊艳的 / 史诗级的 / 超高质量的 / 电影感十足的 / 美轮美奂 / 高级感 / 8K / 超真实`
- 英文：`stunning / cinematic / epic / masterpiece / 8K / ultra-real / amazing / beautiful`

替换方式示例：

- `电影感灯光` → `单侧硬光 45° 打在脸上，另一侧基本落入阴影`
- `史诗感战斗` → `远景交代上百名士兵在泥地厮杀，长焦压缩画面，关键打击瞬间使用 0.3× 升格`
- `美丽的日落` → `太阳刚刚落下地平线 5 分钟，天空呈橙红到深蓝渐变，人物只剩剪影`

优化顺序（超长时优先删什么）：

1. 先删所有「好看/厉害」的形容词（beautiful/stunning/epic…）
2. 压缩环境描述到 3 词以内（如 `雾气山路 / 夜晚街口`）
3. 保留 1–2 个 style token，其余砍掉
4. 如仍超长，优先删声音层，而不是主体/动作/镜头
