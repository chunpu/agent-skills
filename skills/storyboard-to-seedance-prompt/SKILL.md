# 文字分镜到 Seedance2.0 提示词

## 目标
- 把每个 clip 的文字分镜，转化为更具体、更可执行的 Seedance 2.0 视频提示词，并将产物写到文字分镜文件的同目录。

## 输入输出
- 输入：`文字分镜.txt`（含本集所有 clip）或者一句话要求
- 输出：与输入文件同目录的 `视频_ClipXXX.prompt.txt`（每个 clip 一个）和对应视频 Clip。

## Workflow
- 查看用户指定的 `文字分镜.txt`，按 clip 划分并逐条读取分镜内容，不得跳段或合并。
- 为每个 clip 在该 `文字分镜.txt` 的同目录新建/更新 `视频_ClipXXX.prompt.txt`，先写“场景与时间”，再写“分镜”时序段，默认快节奏切分并覆盖 0–15s 全时长。
- 对照“硬性标准/自检清单”逐条补全缺失信息，直到满足最低信息线与时序完整性。
- 确认每个 `视频_ClipXXX.prompt.txt` 的最后一行是“不要BGM，不要字幕”。
- 生成视频：使用 jimeng-skill 的 `multimodal2video` 从 `视频_ClipXXX.prompt.txt` 读取提示词生成 clip，并在生成参数中传入比例（16:9/9:16）、模型（Seedance 2.0）以及角色/场景/道具参考图清单以保持一致性。

## 使用方法
- 针对文字分镜的每个 clip，按下述脚手架填充后写“执行指令段”，文件末尾固定追加“不要BGM，不要字幕”。
- 提示词文本只写画面与执行细节，不写比例、模型、参考图清单等生成参数。

## 执行指令段
- 用 3–6 句覆盖全片段时序，每句包含“主体+动作+镜头语言+空间/光线细节+时间点”，从脚手架压缩而来。
- 结尾固定追加：不要BGM，不要字幕。

## 硬性标准
- 不得合并或省略分镜项；必须覆盖原分镜的全部场景与动作。
- 最低信息线：每个 clip ≥10 个具体名词、≥5 个动词、≥3 个镜头术语。
- 禁止使用“漂亮、好看、震撼”等模糊词，改用可度量细节与可执行动作。
- 明确比例与模型：16:9 或 9:16；Seedance 2.0。
- 绑定一致性资产：角色/场景/道具参考图路径与出镜顺序、出现次数。

## 自检清单
- 机位/镜头类型/镜头运动是否明确
- 时间点与节奏是否完整
- 构图与空间、光线与色彩、摄影参数是否填写
- 参考图是否绑定且顺序/次数已标注
- 末尾是否追加“不要BGM，不要字幕”

## 生成提示
- 写完同目录的 `视频_ClipXXX.prompt.txt` 后，使用视频生成技能读取该文件进行生成，需传入参考图清单，确保元素一致性与画面连贯性。

## 关联技能
- text-storyboard（输入来源）
- generate-film-video-prompt（写作细节参考）
- jimeng-skill（视频生成，使用这个技能生成视频）

## 示例（视频提示词）

```
雨夜的老城石板巷道，路灯昏黄，右上角红蓝霓虹断续闪烁，地面积水形成镜面，空气湿冷

分镜:
0–1.5s: 大全景低角度建立环境，雨线斜向下，路灯在地面形成暖黄光圈，右上红蓝霓虹在水坑镜面出现条状反射；镜头快速轻推至中景。
1.5–3s: 中景平视推近至近景，主角抬头望霓虹，伞沿雨滴成串落下，面部被冷蓝霓虹主光包裹，路灯在面部右缘形成暖色边光，呼吸起伏可见。
3–4.5s: 近景微仰跟拍，脚步踩入水坑溅起 10–15cm 水花，袖口甩动带出水滴拖影，背景远处警笛光带掠过一次，景深浅化突出紧张眼神。
4.5–6s: 眼睛极致特写，水滴沿睫毛滑落，瞳孔轻微收缩，霓虹的红蓝色块在瞳孔中反射；微摇镜捕捉面部细微表情变化。
6–7.5s: 中景侧移，人物由慢转快，镜头与其同向平移，前景路人模糊掠过形成自然遮挡，三分法稳定构图，空间层次明确。
7.5–9s: 低角度大全跟拍，黑伞半收高举，风衣下摆被风掀起；镜头短环绕约 30°，强调紧张态势与环境压迫。
9–10.5s: 近景手部特写，伞柄握紧到指节泛白，雨水沿伞柄滴落形成连续滴线；快门略慢，手部运动呈轻微拖影。
10.5–12s: 中景反打，巷口拐角在前方，地面霓虹反光形成对角线引导；镜头拉远到全景，路灯暖黄将空间切分出明暗块面。
12–15s: 全景静止收束 1s，主角拐角消失，只余雨幕与霓虹反射；前景旗板遮挡制造暗角，最后 1s 保持静止，雨声仅以画面暗示。
不要BGM，不要字幕
```

## 生成参数
- 生成视频时再通过视频生成技能传入比例（16:9/9:16）、模型（Seedance 2.0）与参考图清单（角色/场景/道具路径），这些不写进提示词文本。
