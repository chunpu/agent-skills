# Agent Skills

中文版 | [English](./README.md)

AI 智能体技能合集，包括即梦 (Jimeng) 技能、影视创作、图片视频生成、小说处理等。

## 安装

使用以下命令将这些技能添加到你的 AI 智能体：

```bash
npx skills add chunpu/agent-skills -y -g
```
